# Linux 进程学习

一个命令 一个进程     一个命令运行两次都是不同的进程 有不同的PID

```shell

ps 	 
-e   显示所有进程的信息 
-f 	 显示更加详细的信息
常用 pe -ef 
	
pstree	以树状图形式显示进程之间的关系


kill 
-9 强制结束进程
-stop 挂起进程

pkill  结束一组同名的进程
pkill sleep   杀掉所有名为 sleep命令的进程

& 后台运行程序

jobs显示终端当前任务

fg bg
fg %任务号 将后台运行的或挂起的进程挪到前台执行 
bg %任务号 将挂起的进程唤醒到后台执行


top

```
